Overview

MozSecWorld integrates the three password safety measures: password choice safety, passowrd storage safety, and online password attack prevention.

The demo is going to be performed on the [login] and [registration] pages. Please read the demo section to know what to do.

View the code section to see the code used to implement the three password safety measures for MozSecWorld.

Password Choice Safety:
Try registering with a common password like, “1234567”, or a password fewer than 6 characters. MozSecWorld will prevent you from selecting this weak password.

Sorry, there is no easy way to demo it. Take a look at the [code] and a blog post titled SHA 512 w/ per User Salts is Not Enough.
